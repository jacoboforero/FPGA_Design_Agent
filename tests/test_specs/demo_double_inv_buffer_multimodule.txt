Module: double_inv_buffer_top
Title: Simple multi-module double-inverter buffer demo

Functional intent
Role summary: Top-level integration module that instantiates inv_stage1 and inv_stage2 to compute y = a through two inversions.
Key rules:
- inv_stage1 computes mid_n = ~a.
- inv_stage2 computes y = ~mid_n.
- double_inv_buffer_top performs only structural integration and explicit wiring between child modules.
Performance intent: Same-cycle combinational response from input to output.
Reset semantics: No reset; output depends only on current input.
Corner cases:
- a=0
- a=1
Open questions:
- none

  direction: INPUT

  semantics: top-level input

  direction: OUTPUT
  width_expr: 1
  semantics: top-level output equal to a after two inversions
Handshake semantics:

  rules: none
Transaction unit: one combinational evaluation of y from a.
